Changing the Author of a Commit in Git

To change the author of a commit, you can use the git commit --amend command with the --author flag. Here's how you can do it: 

        git commit --amend --author="New Author Name <email@example.com>"

Replace "New Author Name <email@example.com>" with the new author's name and email address.

In your case, since you want to change the author of a specific commit, you can use the following steps:

  1. Identify the commit hash of the commit you want to change the author for (which you've already done).
  2. Use the git rebase -i command to start an interactive rebase session.
  3. Replace pick with edit next to the commit you want to change.
  4. Amend the commit using the git commit --amend --author="New Author Name <email@example.com>" command.
  5. Continue the rebase with git rebase --continue.
  6. If you've already pushed the commit you've modified, you may need to force-push the changes with git push --force.

This should change the author of the specified commit.
